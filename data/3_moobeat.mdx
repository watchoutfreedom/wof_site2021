<h4>Moobeat is an open software developed with Unity and PureData which purpose is to play music and make art.</h4>

<p><a href="http://www.moobeat.com">Moobeat</a> is an instru-game. A new app created together with <a href="http://www.noon-interactive.com">Noon</a> to make art through body movements. Music, paintings and sculpture controlled using any motion capture camera.</p>

<iframe width="100%" height="450" src="https://www.youtube.com/embed/rs4w7ZXfc0s" frameborder="0" allowfullscreen></iframe>

<p>It was born in 2012 and has since then been selected by :</p> <h3><a href="https://sonar.es/">Sonar '14</a></h3><h3><a href="http://www.zincshower.com/">Zinc Shower '14</a></h3><h3><a href="http://www.blogyuzz.org/2015/02/moobeat-mejorar-a-base-de-musica-conoce-al-iii-ganador-yuzz-2014/">Yuzz</a><a href="http://www.blogyuzz.org/2015/02/moobeat-mejorar-a-base-de-musica-conoce-al-iii-ganador-yuzz-2014/">'14</a></h3> 
<p>and other events as an innovative music creation tool. The software is now available <a href="http://www.moobeat.com">online</a> so anyone can download and try it at home (you need a Kinect 1 camera). Next step?</p>
